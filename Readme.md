__node-crash__ crashes node platform intentionally. __Only for experimental purposes__. DO NOT EVIL.


## Installation

npm:

    $ npm install crash

## Usage

JavaScript Example:

```js
var crash = require('crash');

crash.crash();  // crash occured!
```

Command line example:

    $ crash
